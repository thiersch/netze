# Computernetze

Ein **Computernetz** oder **Rechnernetz** ist ein Verbund **elektronischer Systeme**, die so gekoppelt sind, dass sie miteinander kommunizieren können. Solche elektronischen Systeme sind in der Regel Computer, Sensoren und Aktoren. 

<Columns> 
  <Column className='text--left'>
    #### Computer
    Geräte, die anhand von Rechenvorschriften (Algoritmen) Daten verarbeiten.


    Merkhilfe: englisch _to compute_, rechnen.
  </Column>
  <Column className='text--left'>
    #### Sensoren
    Technische Bauteile, die einen Messwert aus der Welt erfassen und in ein Signal umformen, das weiterverarbeitet werden kann. 

    Merkhilfe: englisch _to sense_, fühlen.
  </Column>
  <Column className='text--left'>
    #### Aktoren
    Technische Bauteile und Gegenstücke zu Sensoren, sie setzen verarbeitbare Signale in Effekte in der Welt um.


    Merkhilfe: englisch _to act_, handeln.
  </Column>
</Columns>
