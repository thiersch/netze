---
title: Layer 5
---

Gängige Betriebssysteme enthalten die gängigen Protokolle auf Layer 2, Layer 3 und Layer 4. Die Protokolle auf Layer 5 bis Layer 7 werden von Programmen ausgeführt, die wir selbst schreiben können oder die wir installieren. Das heißt, dass Programme wie Webclients (Browser) die für Web nötigen Protokolle auf Layer 5 bis 7 (besonders HTTP und HTTPS) selbst implementieren müssen.
